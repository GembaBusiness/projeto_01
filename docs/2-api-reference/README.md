# 2. Referência da API

Bem-vindo à documentação de referência da nossa API. Esta seção fornece detalhes completos sobre todos os endpoints disponíveis, métodos de autenticação, estruturas de dados e códigos de erro.

## URL Base

Todas as URLs da API referenciadas nesta documentação têm o seguinte URL base:

`https://<project-ref>.supabase.co/rest/v1`

## Versionamento

A nossa API usa versionamento na URL para garantir que as alterações não quebrem as integrações existentes. A versão atual é a **v1**.

- **[v1](./v1/)**: Documentação para a versão 1.
- **v2**: (Ainda não disponível)

## Autenticação

A autenticação é feita através de `Bearer Tokens` no cabeçalho `Authorization`. Consulte a página de [autenticação](./v1/autenticacao.md) para mais detalhes.

## Ferramentas

Recomendamos o uso de ferramentas como o [Postman](https://www.postman.com/) ou [Insomnia](https://insomnia.rest/) para testar os endpoints da API.
