# API Versão 1 (v1)

Esta seção documenta a versão 1 da nossa API.

## O que há de novo na v1?

A v1 é a primeira versão pública da nossa API e inclui as funcionalidades essenciais para a gestão de empresas e utilizadores.

## Endpoints

A documentação para os endpoints está dividida por recurso:

- **[Autenticação](./autenticacao.md)**: Como obter um token de acesso.
- **[Empresas](./endpoints/empresas.md)**: Endpoints para criar, ler, atualizar e apagar empresas.
- **[Utilizadores](./endpoints/utilizadores.md)**: Endpoints para gerir perfis de utilizador.

## Erros

Para uma lista completa dos possíveis códigos de erro e seus significados, consulte a página de [Erros](./erros.md).
