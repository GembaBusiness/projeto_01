# 1. Arquitetura do Sistema

Esta seção descreve a arquitetura de alto nível do projeto. O objetivo é fornecer uma compreensão clara dos principais componentes, como eles se interligam, e as decisões arquiteturais que moldaram o sistema.

## Conteúdo

- **[1.1 Visão Geral](./1.1-visao-geral.md)**: Diagramas (C4 Model) e descrição dos principais blocos de construção do sistema (frontend, backend, base de dados, serviços externos).
- **[1.2 Modelo de Dados](./1.2-modelo-de-dados.md)**: Uma visão geral das principais entidades de dados e seus relacionamentos, sem entrar nos detalhes de cada coluna da tabela.
- **[1.3 Decisões Chave (ADRs)](./1.3-decisoes-chave.md)**: Um registro das decisões arquiteturais mais importantes (Architecture Decision Records). Cada ADR explica o contexto, a decisão tomada e as suas consequências.
