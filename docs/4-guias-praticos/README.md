# 4. Guias Práticos (Tutoriais)

Esta seção fornece guias passo a passo para realizar tarefas comuns e essenciais no projeto. O objetivo é acelerar o processo de onboarding de novos desenvolvedores e facilitar o uso da API por outras equipas.

## Tutoriais Disponíveis

- **[4.1 Configurar o Ambiente Local](./4.1-configurar-ambiente-local.md)**: Um guia completo para clonar o repositório, instalar as dependências e iniciar o projeto localmente.

- **[4.2 Registar a Primeira Empresa e Utilizador](./4.2-registar-primeira-empresa.md)**: Um tutorial que mostra como usar a API (ou as funções da base de dados) para criar uma conta de utilizador, registar uma nova empresa e associar o utilizador a essa empresa.

Se tiver sugestões para novos guias, por favor abra uma issue no repositório.
