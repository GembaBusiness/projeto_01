# Documentação do Projeto

Bem-vindo à documentação oficial do nosso projeto. Este é o lugar central para encontrar todas as informações técnicas, desde a arquitetura de alto nível até os detalhes de cada endpoint da API.

## Organização

A documentação está dividida nas seguintes seções principais:

- **[1-arquitetura/](./1-arquitetura/)**: Decisões de alto nível, visão geral do sistema e diagramas.
- **[2-api-reference/](./2-api-reference/)**: Referência completa da nossa API, incluindo endpoints, autenticação e exemplos.
- **[3-base-de-dados/](./3-base-de-dados/)**: Detalhes sobre o esquema do banco de dados, tabelas e funções.
- **[4-guias-praticos/](./4-guias-praticos/)**: Tutoriais passo a passo para realizar tarefas comuns.
- **[5-infraestrutura-e-deploy/](./5-infraestrutura-e-deploy/)**: Informações sobre o ambiente de desenvolvimento, staging, produção e o processo de CI/CD.
- **[GLOSSARIO.md](./GLOSSARIO.md)**: Um dicionário de termos e conceitos específicos do projeto.

Use o menu de navegação ou os links acima para explorar as seções.
